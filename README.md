# Crypto Tracker (React + Vite)

A component-based crypto tracker built with React that shows live market data, supports searching coins, and provides a clean UI with dark/light theme.

## Features
- Live crypto prices (USD)
- Search coins by name
- Auto-refresh prices (every 30 seconds)
- Coin details modal (price, market cap, 24h high/low)
- Dark/Light theme toggle
- Component-based architecture (clean separation of UI)

## Tech Stack
- React
- Vite
- JavaScript (ES6+)
- CSS
- CoinGecko API

```bash
npm install
npm run dev
